# 📘 Ejercicios de JavaScript – Prácticas con DOM y BOM

Este proyecto contiene una serie de ejercicios prácticos desarrollados con **HTML, JavaScript y Bootstrap**, enfocados en el uso del **DOM (Document Object Model)** y el **BOM (Browser Object Model)**.

El objetivo principal es reforzar conceptos clave de JavaScript mediante la interacción con interfaces web dinámicas.

---

## 🚀 Tecnologías utilizadas

* HTML5
* CSS3 (con Bootstrap)
* JavaScript (ES6+)

---

## 📂 Ejercicios incluidos

### 🎯 1. Números Mágicos (Dificultad: 🟢)

Aplicación donde el usuario debe adivinar un número aleatorio generado por el sistema.

**Funcionalidades:**

* Botón para iniciar el juego
* Generación de número aleatorio
* Input para ingresar el número
* Validación mediante `alert`:

  * Indica si el número es mayor o menor
  * Mensaje de éxito al adivinar correctamente

---

### 👤 2. Generaciones (Dificultad: 🟢🟡)

Implementación de una clase `Persona` con múltiples propiedades y métodos.

**Propiedades:**

* Nombre
* Edad
* DNI
* Sexo
* Peso
* Altura
* Año de nacimiento

**Métodos:**

* `mostrarGeneracion()`: Indica a qué generación pertenece
* `esMayorDeEdad()`: Verifica si es mayor de edad
* `mostrarDatos()`: Devuelve toda la información

**Interfaz:**

* Formulario para crear una persona
* Botones para ejecutar cada método
* Resultados mostrados mediante `alert`

---

### 📝 3. Lista de Tareas (Dificultad: 🟢🟡🔴)

Aplicación para gestionar tareas dinámicamente.

**Funcionalidades:**

* Input para ingresar tareas
* Botón para agregarlas a una lista
* Cada tarea incluye un botón para eliminarla

---

### ⏰ 4. Reloj (Dificultad: 🟢🟡)

Visualización de un reloj en tiempo real.

**Funcionalidades:**

* Muestra hora, minutos y segundos
* Actualización automática con `setInterval`

---

### ⏱️ 5. Cronómetro (Dificultad: 🟢🟡)

Cronómetro funcional con control de tiempo.

**Funcionalidades:**

* Botón iniciar
* Botón pausar
* Botón reset (reinicia a 0)

---

### ⏳ 6. Temporizador (Dificultad: 🟢🟡)

Cuenta regresiva configurable por el usuario.

**Funcionalidades:**

* Input para ingresar el tiempo inicial
* Botones:

  * Iniciar
  * Pausar
  * Reset
* Decremento automático del tiempo

---

## 🧠 Conceptos aplicados

* Manipulación del DOM
* Eventos en JavaScript
* Uso de `setInterval` y `setTimeout`
* Programación orientada a objetos (POO)
* Manejo de formularios
* Creación dinámica de elementos HTML
* Interacción con el usuario mediante `alert`

---

## 📌 Objetivo del proyecto

Afianzar la lógica de programación en JavaScript y comprender cómo interactuar con el navegador utilizando el DOM y el BOM, desarrollando pequeñas aplicaciones funcionales.

---

## ▶️ Cómo ejecutar el proyecto

1. Clonar o descargar el repositorio
2. Abrir el archivo `index.html` en el navegador
3. Interactuar con cada ejercicio desde la interfaz
